## 27. [MITRE ATLAS Matrix](https://atlas.mitre.org/matrices/ATLAS)

**Type:** Knowledge Base / Framework  
**Description:** MITRE ATLAS (Adversarial Threat Landscape for Artificial-Intelligence Systems) is a detailed, living knowledge base modeled after MITRE ATT&CK but focused specifically on adversarial machine learning and AI system threats. It organizes adversary tactics (why) and techniques (how) for attacks against AI/ML systems across their lifecycle, including reconnaissance, model evasion, data poisoning, and more. It helps security teams understand AI-specific attack surfaces to develop defenses and mitigation strategies.  
**Use case:** Framework for threat modeling, defense planning, and adversarial testing of AI and ML systems security, including identity-related AI risks.
